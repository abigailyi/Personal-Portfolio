Weather App is a weather forecast app that will pull daily and weekly weather data via API from OpenWeatherMap. This project was coded in vanilla Javascript, HTML, CSS, and Bootstrap. (ReactJS version coming soon!)


Croissant Boy

Croissant Boy is my first responsive multi-page website that I built as a project during my Responsive workshop with SheCodes. It was coded using HTML, CSS, and Bootstrap.


STAMPD App

STAMPD App is concept for an app created for customer retention via stamp cards that I built as a project during my UX boot camp with Springboard. The intent was to create an app with a centralized wallet to carry various stamp cards for redemption at your favorite local businesses rather than bloat the wallet.
